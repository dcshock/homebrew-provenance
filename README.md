# Dcshock Provenanced

## How do I install these formulae?

`brew install dcshock/provenanced/<formula>`

Or `brew tap dcshock/provenanced` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "dcshock/provenanced"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
